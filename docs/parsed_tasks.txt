Đã cắt
Lý do / Thay thế
RBAC 6 vai trò phức tạp
→ Dùng simple JWT, 1 admin role. RBAC bổ sung sau.
Redis + Celery task queue
→ Dùng Python asyncio đơn giản. Đủ cho demo 1 scan tại thời điểm.
PostgreSQL
→ SQLite. Zero-config, zero-setup, sufficient for demo data volume.
CI/CD pipeline (GitHub Actions)
→ Docker Compose thủ công. Đủ cho demo.
WebSocket real-time progress
→ Polling đơn giản mỗi 3 giây từ frontend.
TOTP 2FA + SSO OAuth
→ Chỉ giữ JWT. 2FA/SSO bổ sung post-demo.
WAF bypass testing module
→ Không có WAF trên demo target → không applicable.
API Security Testing (GraphQL…)
→ Chỉ test REST endpoints cơ bản. GraphQL/WebSocket sau.
Continuous monitoring / scheduler
→ Không cần cho 1-time demo. On-demand scan là đủ.
DNS security, port scanning
→ Out of scope cho demo web app.
Audit logs đầy đủ
→ Chỉ log scan events cơ bản trong SQLite.
